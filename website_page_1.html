Library Database

Goal

- create a library database
- create a view
- create a pivot table & pivot chart using the view

Purpose

The purpose of creating and selecting portions of a database to be filtered out is to store all data, and to be able to quickly answer questions about the data inside the database that are asked most frequently.

View

With the mock data, I can now create a view. A view is a pre selected filter that creates a table with filtered values. The view that I chose to make is one that grabs the expiration date of a library card alongside the user info relevant to that card. I chose this because I believe it would be in the best interest of the library to notify the users that they must renew their card in order to keep the community happy. Views like this allow for improtant data like this to not just be stored in a table, but to be readily accessible to whomever needs to access it.



Python Script

Now that I created my database and made a view to go along with it, the next thing to do is play with the view in Excel. In order to do this, I used Pandas and PowerQuery to make a live connection between my database and an excel sheet. The pandas extension allows me to import my database, while PowerQuery gives excel access to the document.

Excel

Once the view was imported into excel, I made a pivot table and pivot chart to further analyze the selected data. This illustrated and further clarified what the data in the view meant. This pivot chart takes the library cards and who they belong to from the view, and shows that from that pool of library cards, which ones are set to expire by 2026. This can be very useful becasue it would allow the library to send out reminders to the users whom need to revalidate their library card soon.



Takeaway

The main purpose of this project was to design, create, customize, export, and analyze a database meant for a library. This project showed many aspects of a database, and how it is not just a place to store data. A database is more capable than jsut that, it allows for analyses, organization, and efficient retrieval of information, making it a powerful tool for managing complex networks. ##
